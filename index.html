Connect to GitHub

Save edits directly to your repo in one click instead of exporting and uploading manually. Your token is stored only in this browser and sent only to GitHub.

How to create a Personal Access Token (≈ 3 minutes)
  1. Go to github.com/settings/personal-access-tokens/new (sign in if needed)
  2. Token name: MEC Compensation Editor
  3. Expiration: 90 days (or whatever suits you)
  4. Resource owner: ncoulmec
  5. Repository access: Only select repositories → pick mec-compensation
  6. Permissions → Repository permissions → Contents → set to Read and Write
  7. Click Generate token at the bottom
  8. Copy the token (it starts with github_pat_) and paste it above
