📖 Key terms legend
Revenue — total invoiced sales ex-GST (Xero "Total Income").
COGS — direct cost of goods sold (talent fees, freelancers paid per-project, etc.).
GP (Gross Profit) = Revenue − COGS.
Contribution Expenses — direct staff cost attributable to delivering revenue (Sales/Account team salaries + super + commissions).
CM (Contribution Margin) = GP − Contribution Expenses. The cash left after the team that produced the revenue is paid.
OpEx — overheads not tied to a particular project (rent, software, finance, leadership salaries excl. team commission).
NP (Net Profit) = CM − OpEx. The bottom line.
LTM — Last Twelve Months. Rolling sum of the last 4 complete quarters.
YoY — Year-over-Year. Same metric this quarter vs the same quarter last year (from Previous Quarters Historical).

How to create a Personal Access Token (≈ 3 minutes)
  1. Go to github.com/settings/personal-access-tokens/new (sign in if needed)
  2. Token name: MEC Compensation Editor
  3. Expiration: 90 days (or whatever suits you)
  4. Resource owner: ncoulmec
  5. Repository access: Only select repositories → pick mec-compensation
  6. Permissions → Repository permissions → Contents → set to Read and Write
  7. Click Generate token at the bottom
  8. Copy the token (it starts with github_pat_) and paste it above
